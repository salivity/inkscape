How to Adjust Letter Kerning in Inkscape?

Adjusting the spacing between individual letters, known as kerning, is a fundamental skill for creating professional typography and custom logos in Inkscape. While the software automatically handles standard letter spacing, manual kerning allows you to fix awkward gaps between specific character pairs that standard fonts might miss. This guide covers the quickest keyboard shortcuts, the Text tool settings, and how to reset your adjustments if you make a mistake.

Using Keyboard Shortcuts for Fast Kerning

The most efficient way to adjust kerning in Inkscape is by using keyboard shortcuts directly on your canvas. This method allows you to visually tweak the spacing in real-time.

  1. Select the Text tool (or press T) and click to place your cursor between the two letters you want to adjust.
  2. Hold down the Alt key.
  3. Use the Left or Right Arrow keys to move the letter to the right of your cursor closer or further away.

Tip: If you want to make larger adjustments quickly, hold down **Shift + Alt** while pressing the arrow keys to move the letters in larger increments.

Adjusting Kerning via the Tool Control Bar

If you prefer precise pixel measurements or want to adjust the spacing across a selection of text, you can use the numeric fields at the top of the workspace.

  1. Select your text or place your cursor between a specific letter pair.
  2. Look at the Tool Control Bar at the top of the screen.
  3. Locate the Horizontal Kerning field (represented by an icon with an ‘A’ and a ‘B’ separated by a horizontal arrow).
  4. Enter a positive number to increase the space, or a negative number to decrease it.

Using this panel is highly beneficial when you need to match exact design specifications across multiple text elements.

Moving Letters Vertically

Inkscape also allows you to shift individual letters up or down, which can create a dynamic, playful layout or help align complex scripts.

  • Place your cursor before the letter you want to shift.
  • Hold Alt and press the Up or Down Arrow keys.
  • To make larger vertical jumps, use **Shift + Alt** with the vertical arrow keys.

How to Reset Kerning Adjustments

If your experiments go wrong or the spacing looks uneven, you do not need to delete your text and start over. You can easily clear all manual shifts.

  • Select the text block with the Text tool.
  • Go to the top menu and click on Text.
  • Select Remove Manual Kerns from the dropdown menu.

This action instantly restores the font to its original default spacing, allowing you to start fresh.
