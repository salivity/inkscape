How Do You Open a New Document in Inkscape?

This article provides a straightforward guide on how to open a new document in Inkscape. Whether you are starting a fresh vector graphics project from scratch or need to open a template with specific dimensions, we will cover the quick keyboard shortcuts, the standard menu navigation, and the welcome screen options required to get your workspace ready.

Using Keyboard Shortcuts

The fastest way to open a blank document in Inkscape is by using keyboard shortcuts. This bypasses the menus entirely and immediately brings up a fresh canvas.

  • Windows and Linux: Press Ctrl + N
  • macOS: Press Cmd + N

This command will instantly generate a new canvas using your default document properties.

Navigating the Top Menu

If you prefer using your mouse to navigate the graphical interface, you can easily open a new document through the main application menu.

  1. Locate the top-left corner of your Inkscape window.
  2. Click on the File menu.
  3. Select New from the dropdown list.

Choosing a Specific Template

Sometimes you need a document with precise dimensions, such as a standard letter page, a mobile screen, or a social media banner. Inkscape allows you to open new documents based on pre-defined templates.

  1. Click on the File menu.
  2. Select New from Template… (Shortcut: Ctrl + Alt + N on Windows/Linux, Cmd + Option + N on macOS).
  3. A dialog box will appear. Browse through the categories to find the exact size and format you need.
  4. Click Create from Template to open your new workspace.

Using the Welcome Screen

In recent versions of Inkscape, a Welcome Screen appears when you first launch the application. This screen is designed to help you quickly set up a new document before you even reach the main editing interface.

  1. Launch the Inkscape application.
  2. On the Welcome Screen, look at the Time to Draw tab.
  3. Select your desired format (such as Print, Screen, Video, or Social) from the available options.
  4. Click on the specific size icon, and the new document will open automatically.
